# 防抖节流实现

---

## 一、函数防抖（debounce）实现

``` javascript
function debounce(fn,waitTime){
  let timer;
  return (...args) => {
    clearTimeout(timer);
    timer = setTimeout(() =>{
      fn(...args)
    },waitTime)
  }
}
```


## 二、函数节流 (throttle) 实现

``` javascript
function throttle(fn,waitTime){
  let timer
  return (...args) => {
    if (timer){ return }
    timer = setTimeout(()=>{
      fn(...args);
      timer = null;
    },waitTime)
  }
}
```
