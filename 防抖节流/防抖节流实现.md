# 防抖节流实现

---

## 一、函数防抖（debounce）实现

``` javascript
function debounce(fn,waitTime){
  let timer;
  return (...args) => {
    clearTimeout(timer);
    timer = setTimeout(() =>{
      fn(...args)
    },waitTime)
  }
}
```
